Satellite Image Classification

This project focuses on classifying satellite images into three distinct land-use classes: urban, rural, and forest. The classification is achieved through the application of various unsupervised clustering algorithms.

Objective

To explore and implement different unsupervised clustering methods to classify satellite images based on their land-use types, utilizing a dataset characterized by four key variables.

Methodology

Data Preprocessing

  • Importing Data: Data was imported using the "Excel Reader" node in KNIME.
  • Normalization: The dataset was normalized using the "Normalizer" node.
  • Partitioning: The dataset was split into training and testing sets (70-30%) using the "Partitioning" node.

Clustering Algorithms

Several unsupervised clustering models were tested, including:

  • K-Means Clustering: Groups data into k clusters based on proximity to centroids.
  • Hierarchical Clustering: Constructs a dendrogram to represent proximity relationships.
  • DBSCAN: Identifies high-density regions separated by low-density regions.
  • Fuzzy c-Means: Allows points to belong to multiple clusters.
  • Self-Organizing Map (SOM): Uses a neural network for clustering and dimensionality reduction.

Evaluation Metrics

  • User's Accuracy (UA): The correctness of the model in predicting a category.
  • Producer's Accuracy (PA): The model's ability to identify real instances of a category.
  • Overall Accuracy (OA): The overall accuracy of the model across all categories.

Results

Each clustering model was evaluated through confusion matrices and the calculated accuracy metrics. Two test scenarios were conducted:

  • Using the test dataset obtained through partitioning.
  • Using the entire dataset, including the test set.

The results indicated that the Self-Organizing Map (SOM) and Fuzzy c-Means algorithms were the most effective, with high accuracy and consistency across different test scenarios.
